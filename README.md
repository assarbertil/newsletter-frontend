# Frontend for newsletter app

## Instructions

This repo needs to be hosted on a web serving platform like vercel and the environment variables need to be set according to example below.

## .env shape

```.env
NEXT_PUBLIC_BACKEND_URL="<url>"
```

## Node modules

### Dependencies

- stitches
- axios
- next
- react
- react-dom
- swr

- eslint
